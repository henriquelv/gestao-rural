# Gestão Rural App

Aplicativo mobile desenvolvido em React + Capacitor para gestão de operações rurais e pecuárias.

Este repositório contém todo o código-fonte necessário para rodar a aplicação localmente.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
